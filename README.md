# University of Edinbrugh 3rd year course Informatics large practical

### Starting up the WebServer

1. Go to the directory containing the WebServer
2. `java -jar WebServerLite.jar`

### Running the Java program

- Command Line: `java -jar target/aqmaps-0.0.1-SNAPSHOT.jar 22 12 2020 55.9444 -3.1878 5678 8080`
- Arguments are the following: day, month, year, starting latitude, starting longtitude, seed, and port number of web server
- You can render the content from the geojson file on geojson.io
